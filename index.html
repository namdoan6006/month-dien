📊 Kết nối Google Sheets

Đọc & ghi dữ liệu trực tiếp với Google Sheets. Cần 2 URL: Sheet (để đọc) và Apps Script (để ghi).

○Chưa kết nối
📋 Cấu trúc Google Sheet
Sheet Contracts (sheet đầu tiên):
• Header: name color T1 T2 ... T12
• Mỗi hàng = 1 hợp đồng · cột color: hex vd #f59e0b (có thể để trống)

Sheet Tariff (sheet thứ 2, tùy chọn):
• Header: name vat limit rate
• Hàng 1: tên biểu giá, VAT% (vd: 10), để trống, để trống
• Hàng 2+: (trống) (trống) 50 1984 → mỗi hàng 1 bậc giá
⚙️ Cách tạo Apps Script (để ghi dữ liệu)
1. Mở Google Sheet → Extensions → Apps Script
2. Xóa code cũ, dán đoạn script bên dưới vào
3. Deploy → New deployment → Type: Web app
   • Execute as: Me
   • Who has access: Anyone
4. Copy URL dạng https://script.google.com/macros/s/.../exec
5. Dán URL vào ô Apps Script phía trên

🔓 Cách public Sheet để đọc:
Share → Anyone with the link → Viewer
